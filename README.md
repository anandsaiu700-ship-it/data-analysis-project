# 📊 Student Data Analysis Project

## 📌 Overview

This project analyzes student performance using Python. It calculates total marks, average marks, and identifies top-performing students.

## 🧠 Features

* Calculate total and average marks
* Identify top 5 students
* Filter students with high performance
* Subject-wise average calculation
* Pass/Fail classification

## 🛠️ Technologies Used

* Python
* Pandas

## 📊 Output

* Updated dataset with Total, Average, and Result
* Insights about student performance

## 🚀 How to Run

1. Install required library:
   pip install pandas openpyxl

2. Run the script:
   python analysis.py

## 📁 Files

* student_data.xlsx → original data
* updated_student_data.xlsx → processed data
* analysis.py → main code
